# 指標と出力

## 現在の指標

現時点の `output/vtracer/experiments/report.md` と CSV が含む主な値：

- 実行時間（秒）
- SVG サイズ（バイト）
- SVG パス数
- fill（色）数
- 出力サイズ（幅 / 高さ）
- 生成ファイル名

処理効率や構造的複雑度を比較するのに適した最小指標です。

## CSV の見方

`output/vtracer/experiments/experiment_summary.csv` は画像ごと / プリセットごと / 条件ごとに 1 行を持ちます。

おすすめの読解順:

1. 実行時間とサイズで効率の高い設定を確認
2. fill / path を見てトポロジーの複雑さを確認
3. Markdown の SVG プレビューで見た目の品質を合わせて確認

## 現在の限界

- 忠実度を評価する画像比較指標は未導入
- 編集しやすさに関する指標は未導入
- ハードウェアを跨る正規化は未実装

既知の制約を明示し、新規指標の優先順位を明確にすることで改善を進めます。
